## 📖 About The Project

<img src="Booken/assets/images/desktop (3).png" alt="Booken Book Website Desktop Screenshot" width="100%" />

**Booken** is a modern and minimalist book website UI created to showcase **books, authors, and collections** in a visually balanced design.  
It features an organized layout, soft color palette, and user-friendly structure — perfect for **bookstores**, **reading clubs**, or **eBook platforms**.  
This front-end concept highlights readability and smooth navigation with a professional and aesthetic look.
